#### Prerequisites

##### Programming language runtime

- **.NET 6+ or newer**

##### Required Dependencies

| Package | Version | Purpose |
|---------|---------|---------|
| `dotenv.net` | 4.0.0 | Environment variable management from .env files |
| `Newtonsoft.Json` | 13.0.4 | JSON serialization/deserialization |

To install these dependencies, execute the below command.

```sh
dotnet restore

dotnet build
```
