#### Prerequisites

##### Programming language runtime

- **PHP 7.4+ or newer**
- **Composer** - For dependency management

##### Required Dependencies

| Package | Version | Purpose |
|---------|---------|---------|
| `vlucas/phpdotenv` | ^5.5 | Environment variable management from .env files |

To install these dependencies, create a `composer.json` file in project root with the dependencies and then execute the below command.

```sh
composer install
```
