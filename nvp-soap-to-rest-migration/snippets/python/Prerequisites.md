#### Prerequisites

##### Programming language runtime

- **Python 3.7+ or newer**
- **pip** - For dependency management

##### Required Dependencies

| Package | Version | Purpose |
|---------|---------|---------|
| `requests` | (latest) | HTTP library for API requests |
| `python-dotenv` | (latest) | Environment variable management from .env files |

To install these dependencies create a `requirements.txt` at project root and then use the below command.

```sh
pip install -r requirements.txt
```
