#### Prerequisites

##### Programming language runtime

- **Node.js 14+ or newer**
- **npm** - For dependency management

##### Required Dependencies

| Package | Version | Purpose |
|---------|---------|---------|
| `axios` | ^1.11.0 | HTTP client for API requests |
| `dotenv` | ^17.2.1 | Environment variable management from .env files |

To install these dependencies, create a `package.json` file in project root with the dependencies and then execute the below command.

```sh
npm install
```
