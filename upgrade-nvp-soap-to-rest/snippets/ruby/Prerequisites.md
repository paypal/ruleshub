#### Prerequisites

##### Programming language runtime

- **Ruby 2.7+ or newer**
- **bundler** - Manage dependencies

##### Required Dependencies

| Gem | Version | Purpose |
|-----|---------|---------|
| `dotenv` | ~> 3.1 | Environment variable management from .env files |
| `json` | ~> 2.7 | JSON parsing and generation |

To install these dependencies, create a `gemfile` at project root and then run the below command.

```sh
bundle install
```
